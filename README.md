# OpenClaw E-Commerce Skills

A collection of [OpenClaw](https://openclaw.ai) skills for e-commerce and online business automation.

## Skills

### ðŸ“¦ [shipstation-orders](./shipstation-orders/)
Monitor ShipStation for new orders and issues. Aggregates orders from multiple marketplaces (Amazon, Etsy, Shopify, TikTok, etc.).

**Features:**
- New order notifications
- Alert for orders stuck in processing (>48h)
- Flag orders on hold
- Daily summary reports
- Automatic state tracking

**Install:**
```bash
clawhub install shipstation-orders
```

## Installation

Install individual skills via [ClawHub](https://clawhub.com):

```bash
clawhub install shipstation-orders
```

Or clone this repo for development:

```bash
git clone https://github.com/cprice70/openclaw-ecommerce-skills.git
cd openclaw-ecommerce-skills
```

## Requirements

- [OpenClaw](https://openclaw.ai) installed
- Node.js (included with OpenClaw)
- Platform-specific credentials (ShipStation, Shopify, etc.)
